4. Outbound links & downloads

Anchors with a host that differs from this page emit an outbound_link event. Anchors ending in known download extensions (or carrying the download attribute) emit a download event. Both are detected via a single delegated click listener.

6. Identifying users

The identifier is hashed with SHA-256 in the browser before transport. Use any opaque value — email, user id, hash. Subsequent events are linked to the resulting user id.

8. Drop-in <script> tag

Most production sites can use the script-tag form, no JavaScript required:

<script defer
  src="https://cdn.jsdelivr.net/npm/oximetric/dist/oximetric.min.js"
  data-server="https://analytics.example.com"
  data-token="pk_xxx"
  data-auto-track="pageviews,outbound,downloads,forms"
  data-consent="granted"
  data-storage="auto"
  data-cookie-domain="auto"
  data-log-level="warn"></script>
